# DezignPool Website v1

Premium Architecture & Interior Design Website built with React, TypeScript, and Tailwind CSS.

## Features

- Modern, responsive design
- Optimized performance
- SEO friendly
- Interactive project showcase
- Contact form integration
- Video testimonials
- Animated UI components

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- EmailJS
- Vite

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Create a `.env` file with required environment variables:
```env
VITE_EMAILJS_KEY=your_emailjs_key
```
4. Start development server:
```bash
npm run dev
```
5. Build for production:
```bash
npm run build
```

## Project Structure
